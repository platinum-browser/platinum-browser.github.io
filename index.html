Privacy by design


The Platinum browser has been designed from the ground up to provide a privacy-focused experience. The only server that Platinum connects to is the update notifier, with no private information sent.


Open Source

All Platinum code is free and open source for anyone to inspect and review. It is all licensed under MIT and anyone can review all of the code.


Modern


Platinum is based on Electron, which supports almost all features of Chromium with peace of mind. Platinum also has an elegant user interface with no unnecessary distractions.
